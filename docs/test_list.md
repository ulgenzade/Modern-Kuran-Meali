# KuranMealApp - Temizlik Sonrası Test Listesi

## 1. Veritabanı ve Performans
- [ ] Uygulama ilk açılışta çökmüyor.
- [ ] Veritabanı (kuran.gz) başarıyla kopyalanıp açılabiliyor.
- [ ] Sureler ve ayetler hızlı yükleniyor.

## 2. Arayüz ve Navigasyon
- [ ] Sureler listesinde kaydırma pürüzsüz.
- [ ] Özel kaydırma çubuğu (scrollbar) titreme olmadan çalışıyor.
- [ ] Sureye tıklandığında ayetler sayfasına sorunsuz geçiliyor.
- [ ] Geri tuşuna basıldığında bir önceki ekrana dönülüyor (çökme yok).

## 3. Okuma ve Alt Paneller
- [ ] Nüzul Sebebi ve Tefsir popupları açılıyor ve kapanıyor.
- [ ] Bottom Sheet panelleri sürükle-kapat mekanizması sorunsuz çalışıyor.
- [ ] Dikey ve Yatay okuma modları arası geçiş çalışıyor.
- [ ] Yazı tipi (Font) değiştirildiğinde ekran anında güncelleniyor.

## 4. Gelişmiş Özellikler
- [ ] Arama sayfasında arama sonuçları listeleniyor.
- [ ] Fihrist sayfası kategorileri açılıp kapanıyor.
- [ ] "Okumaya Devam Et" FAB butonu doğru sure/ayete yönlendiriyor.
